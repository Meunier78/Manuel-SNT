<p class="scrap-without-first-image"></p><div id="note-text"><p>Une liste d'éléments en python s'écrit :<br/></p>
<p><i>une_liste = ['a', 'b', 'c'] </i>avec 0, 1, 2 les indices respectifs de a, b et c. On peut obtenir le deuxième élément avec :</p>
<p><i>une_liste[1]</i> renvoie b</p>
<p>On peut obtenir l'indice du troisième élément (c) avec :</p>
<p><i>une_liste.index('c')</i> renvoie 2</p>
<p>Le maximum d'une liste peut être déterminé avec la fonction <b>max</b>, que les éléments soient des nombres ou des lettres :</p>
<p><i>max_liste = max(une_liste)</i> renvoie 'c' qui est la plus grande lettre dans l'ordre alphabétique.</p>
<p>Pour donner une valeur entière comprise entre deux autres valeurs, on utilise <b>randint</b> du module <b>random</b> que l'on importe avec :</p>
<p><i>from random import randint</i></p>
<p>Il suffit ensuite d'appeler la fonction :</p>
<p><i>nb_aleatoire = randint(100, 200)</i> pour obtenir un nombre entre 100 et 200.</p>
