# Présentation

<p class="scrap-without-first-image"></p><div id="note-text"><p>Nous vous proposons, au travers cette activité, de comprendre comment les pages web peuvent être classées par votre moteur de recherche.</p>
<p>La plupart des pages web présentent des liens vers d’autres pages web, appelés liens hypertextes. Le principe général est que si le lien vers une page web est présent dans beaucoup d'autres pages web, cette page est considérée comme "importante" ou "populaire" et sera bien classée par les moteurs de recherche.<br/></p>
<p>Pour mesurer cette popularité par un nombre, on applique l’algorithme suivant :<br/></p>
<ul><li>on part d’une page web au hasard (on suppose que cela n’a pas d’importance, ce qui est partiellement vrai) ;</li><li>sur chaque page visitée, on : clique sur un lien au hasard sur cette page (pour visiter une nouvelle page) ;</li><li>on répète cette étape un grand nombre de fois.</li></ul><p>Durant l’exécution de l’algorithme, on n’oublie pas de compter le nombre de fois que chaque page a été visitée. Le score de chaque page est alors la fréquence du nombre de visites de cette page. Les pages avec le plus haut score seront visitées plus souvent que les autres.<br/></p>
<p>Ce concept est celui du <u>surfeur aléatoire</u>.</p>
<p><b>Exemple</b><br/></p>
<p>Le graphe "Représentation de pages web" représente un ensemble de page web, nommées A à F.<br/>Les arêtes (flèches) représentent des liens entre les pages. Il y a une arête allant de la page B à la page F, ce qui signifie qu’il est possible, en visitant la page B, de cliquer sur un lien pour se retrouver sur la page F (ignorer les numéros pour le moment).</p>
