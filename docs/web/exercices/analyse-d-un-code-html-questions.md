<p>Observez le code HTML ci-dessus et répondez aux questions suivantes.</p>
<p>1. Quel élément de balise HTML est utilisé pour lier une feuille de style externe à une page web ?<br/></p>
<p>a) style<br/>b) link<br/>c) head<br/>d) script</p>
<p>2. Quel attribut est utilisé pour spécifier le chemin d'accès à une feuille de style externe ?<br/></p>
<p>a) href<br/>b) src<br/>c) alt<br/>d) style</p>
<p>3. Quel élément de balise HTML est utilisé pour créer un lien hypertexte interne ou externe ?<br/></p>
<p>a) url<br/>b) link<br/>c) href<br/>d) a<br/></p>
<p>4. Quel attribut est utilisé pour spécifier le chemin d'accès à une image ?<br/></p>
<p>a) alt<br/>b) href<br/>c) src<br/>d) target<br/></p>
<p>5. Quel élément de balise HTML est utilisé pour créer une liste à puces ?<br/></p>
<p>a) li<br/>b) ui<br/>c) ol<br/>d) a<br/></p>
<p>6. Quel élément de balise HTML est utilisé pour créer un titre de niveau 1 ?<br/></p>
<p>a) p<br/>b) ol<br/>c) ui<br/>d) h1</p>
<p>7. Quel élément de balise HTML est utilisé pour créer un paragraphe ?<br/></p>
<p>a) h1<br/>b) head<br/>c) body<br/>d) p</p>
<p>8. Quel élément de balise HTML est utilisé pour encapsuler les métadonnées ?</p>
<p>a) body<br/>b) html<br/>c) head<br/>d) li</p>
<p>9. Quel élément de balise HTML encapsule le contenu de la page web ?</p>
<p>a) body<br/>b) head<br/>c) html<br/>d) ui</p>
<p>10. Quel est l'élément associé à la balise HTML racine ?</p>
<p>a) head<br/>b) body<br/>c) html<br/>d) p</p>
<p>11. La balise présente sur la première ligne du fichier HTML permet au navigateur de comprendre que les instructions qui vont suivre sont en :</p>
<p>a) HTML 5<br/>b) JSON<br/>c) CSS<br/>d) JAVASCRIPT</p>
