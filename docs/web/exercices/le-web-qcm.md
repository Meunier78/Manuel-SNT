# Le Web - QCM

<p class="scrap-without-first-image"></p><div id="note-text"><p>1. Qu'est-ce que le World Wide Web (WWW) ?<br/></p>
<p>a) Un système distribué de documents interconnectés accessibles via Internet.<br/>b) Un réseau social populaire.<br/>c) Un protocole de transfert de fichiers.<br/>d) Un langage de programmation pour créer des sites web.</p>
<p>2. Qu'est-ce que le HTML ?<br/></p>
<p>a) Un langage de programmation pour créer des sites web interactifs.<br/>b) Un langage pour interroger des bases de données.<br/>c) Un langage de balisage pour structurer le contenu des pages web.<br/>d) Un format de fichier pour stocker des images.</p>
<p>3. Qu'est-ce que le CSS ?<br/></p>
<p>a) Un langage de programmation pour créer des fonctionnalités dynamiques dans les pages web.<br/>b) Un protocole de communication sécurisé.<br/>c) Un langage pour décrire la logique et les fonctionnalités des sites web.<br/>d) Un langage de balisage pour styliser et mettre en forme les pages web. </p>
<p>4. Qu'est-ce que le HTTP ?<br/></p>
<p>a) Un langage de programmation utilisé pour créer des applications web.<br/>b) Un format de fichier pour stocker des vidéos.<br/>c) Un protocole de communication utilisé pour transférer des données sur le web.<br/>d) Un langage de balisage pour structurer le contenu des pages web.</p>
<p>5. Qu'est-ce que le DNS ?<br/></p>
<p>a) Un langage de programmation pour créer des animations sur les sites web.<br/>b) Un service permettant de traduire les noms de domaine en adresses IP.<br/>c) Un protocole de sécurité utilisé pour chiffrer les communications sur le web.<br/>d) Un format de fichier pour stocker des feuilles de style.</p>
<p>6. Qu'est-ce qu'une URL ?<br/></p>
<p>a) Une adresse unique qui identifie une ressource sur le web.<br/>b) Un outil pour créer des diagrammes et des schémas dans les pages web.<br/>c) Un langage de programmation pour interagir avec des bases de données.<br/>d) Un format de fichier pour stocker des scripts JavaScript.<br/></p>
