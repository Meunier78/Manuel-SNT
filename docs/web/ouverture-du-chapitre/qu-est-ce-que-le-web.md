# Qu'est-ce que le Web ?

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Le Web est un système mondial de publication et d'accès à des informations <i>via</i> Internet. Il permet, entre autres, de partager des documents multimédias, des pages web et d'autres ressources. Le Web repose sur les protocoles HTTP et HTTPS et utilise des liens hypertextes pour relier les informations entre elles.<br/></p>
<p>Il permet aux utilisateurs de naviguer entre différentes pages en utilisant des navigateurs web tels que Mozilla Firefox, Google Chrome, Microsoft Edge, Apple Safari, etc. Le Web est composé de sites web hébergés sur des serveurs accessibles <i>via</i> des adresses URL. Il offre des services variés tels que le courrier électronique, le commerce en ligne et les réseaux sociaux.<br/></p>
<p>Le Web est une infrastructure essentielle pour la communication, l'échange d'informations et la collaboration à l'échelle mondiale.</p>
