<p>Le premier site Web créé au CERN, et dans le monde, était destiné au projet World Wide Web lui-même. Il était hébergé sur l’ordinateur NeXT de Tim Berners-Lee. En 2013, le CERN a entrepris de <a href="https://first-website.web.cern.ch/" rel="noopener" target="_blank">remettre en service ce premier site Web</a> : <a href="https://home.cern/fr/science/computing/info.cern.ch" rel="noopener" target="_blank">info.cern.ch</a>.</p>
<p>Le 30 avril 1993, le CERN a mis le logiciel du World Wide Web dans le domaine public. Puis il a émis une version suivante de l’application sous licence libre, une façon plus sûre de maximiser sa diffusion. Ce faisant, il a permis à la Toile de se tisser.</p>
