# Qu'est-ce que Flexbox ?

<div class="imported-content">

<p>Flexbox, également connu sous le nom de CSS Flexible Box Layout, est un module CSS qui permet de créer des mises en page flexibles et réactives. Il offre un moyen efficace de distribuer, aligner et organiser les éléments d'une interface utilisateur, en particulier dans le contexte d'un conteneur parent.</p>
<p>Le principe de base de Flexbox repose sur la création d'un conteneur flexible (<i>flex container</i>) qui contient un ou plusieurs éléments enfants (<i>flex items</i>). Les éléments enfants peuvent être disposés horizontalement ou verticalement, en fonction des besoins de la conception.</p>
<p>Flexbox est largement utilisé dans la conception et le développement web pour créer des mises en page responsives et adaptatives. Il offre une alternative puissante et flexible aux méthodes traditionnelles de mise en page en utilisant des flottants et des positionnements absolus.</p>
