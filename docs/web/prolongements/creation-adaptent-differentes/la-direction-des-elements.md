# La direction des éléments

<div id="note-text"><p><img ="xl"="" class="imported-image ptSynthesisImg" data-height="631" data-original-url="logo%2Fhash%2F551774484%3Fh%3De9fc9b5f05e05d0ed556f7618b27ae0d" data-width="1000" id="l_2432889" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-prolongements-creation-adaptent-differentes-la-direction-des-elements-html/image-01.jpg"/></p>
<p>Flexbox utilise deux axes pour disposer les éléments. L'axe principal (main axis) peut être horizontal ou vertical, selon la direction souhaitée. L'axe transversal (cross axis) est perpendiculaire à l'axe principal.<br/></p>
<p>La propriété <b>flex-direction</b> permet de spécifier la direction de l'axe principal. Elle peut prendre les valeurs suivantes :</p>
<ul><li><b>row : </b>Les éléments sont disposés horizontalement de gauche à droite.</li><li><b>column : </b>Les éléments sont disposés verticalement de haut en bas.</li><li><b>row-reverse : </b>Les éléments sont disposés horizontalement de droite à gauche.</li><li><b>column-reverse :</b> Les éléments sont disposés verticalement de bas en haut.</li></ul><p>Remarque : par défaut avec <b>flex</b>, les éléments sont alignés horizontalement ; il n’est donc pas nécessaire de préciser dans le fichier style.css l’instruction <b>flex-direction : row</b>.</p>
