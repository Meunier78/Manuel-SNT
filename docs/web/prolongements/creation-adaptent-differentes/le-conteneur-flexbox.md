<div id="note-text"><p><img ="xl"="" class="ptSynthesisImg imported-image" data-height="836" data-original-url="logo%2Fhash%2F551774485%3Fh%3D30e15360fba1d7e5c573ea7609c3d1a8" data-width="1000" id="l_2432886" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-prolongements-creation-adaptent-differentes-le-conteneur-flexbox-html/image-01.jpg"/></p>
<p>Le principe de la mise en page avec Flexbox est très simple : on définit un conteneur à l’intérieur duquel on place plusieurs éléments. Une page web peut accueillir plusieurs conteneurs.</p>
<p>Le conteneur est une balise HTML basée par exemple sur l'élément <b>div</b> , et les éléments sont d’autres balises HTML de type <b>div</b> placées à l’intérieur.</p>
<p>Pour activer Flexbox, vous devez appliquer la propriété CSS <b>display: flex</b> à l'élément conteneur.<br/><br/></p>
