<p>Le fichier JavaScript (<b>script.js</b>) contient le code pour manipuler le formulaire. Il récupère le formulaire à l'aide de <b>document.getElementById("monFormulaire")</b> et ajoute un événement de soumission en utilisant <b>addEventListener</b>.</p>
<p>À l'intérieur de la fonction de gestion de l'événement, le code récupère les valeurs saisies dans les champs du formulaire en utilisant les IDs correspondants (nom et email). Il valide ensuite les données et affiche une alerte si des champs sont vides ou traite les données en affichant une autre alerte avec les valeurs saisies.</p>
<p>Grâce à la séparation du code JavaScript dans un fichier distinct, le code HTML reste plus clair et plus concis, et il est plus facile de maintenir et de gérer le code JavaScript de manière indépendante.<br/></p>
