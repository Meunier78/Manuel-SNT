<p>Dans l'exemple ci-dessus, le code JavaScript est écrit entre les balises associées à l'élément <b>script</b> dans la section <b>head</b> du fichier HTML. Il peut contenir des fonctions, des variables, des événements, etc. Ici, une fonction <b>afficherMessage()</b> est appelée lorsque l'utilisateur clique sur le bouton. Cette approche est simple et pratique pour de petites quantités de code. Cependant, elle peut devenir difficile à gérer lorsque le code JavaScript devient plus complexe ou lorsque l'on souhaite réutiliser le code sur plusieurs pages.</p>
