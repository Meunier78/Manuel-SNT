# Ceci n'est pas Internet - Questions

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Après avoir visionné la vidéo "Ceci n'est pas Internet !", répondez aux questions suivantes.</p>
<p>1. Expliquez brièvement ce qu'est Internet.<br/></p>
<p>2. Qu'est-ce que le www ?</p>
<p>3. Quels sont les deux langages utilisés pour coder les informations contenues dans une page web ?</p>
<p>4. Quelle application permet d'afficher des pages web sur un ordinateur, une tablette ou un smartphone ?</p>
<p>5. Qu'est-ce qu'un lien hypertexte ?</p>
<p>6. Qu'est-ce que le Web ?</p>
<p>7. En résumé, quelle distinction peut-on faire entre Internet et le Web ?</p>
<p>8. Qu'est-ce qu'une URL ?</p>
<p>9. Sur quels protocoles s'appuient les URL ?</p>
<p>10. Qu'est-ce qu'un nom de domaine ?</p>
<p>11. Qu'est-ce que le phishing ? Comment s'en prémunir ?</p>
<p>12. Quel type de serveur gère la communication entre un navigateur client et un site web ?</p>
<p>13. Expliquez brièvement les principales étapes permettant l'affichage d'une page web dans un navigateur ?<br/></p>
