# Mozilla Firefox : outils de développement web

<div class="card">




<div id="note-text"><p><img class="ptSynthesisImg imported-image" data-height="406" data-original-url="media%2For%2F551774324%3Fhid%3Dm_2188" data-width="750" id="m_2188" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-activites-requete-http-mozilla-firefox-outils-de-developpement-web-html/image-01.jpg"/></p>
<p>Tous les navigateurs web modernes sont livrés avec un ensemble d'outils de développement. Ces outils permettent de réaliser différentes actions, par exemple :<br/></p>
<ul><li> inspecter le code HTML, CSS ou JavaScript chargé à la volée dans la page ;<br/></li><li>montrer les fichiers téléchargés et leur temps de chargement ;<br/></li><li>etc.</li></ul><p>On accède aux outils de développement du navigateur web Mozilla Firefox en appuyant sur la touche F12.</p>
<p>Le menu "Réseau" permet d'observer, entre autres, les requêtes HTML entre le navigateur et le serveur web en rafraichissant (Ctrl + F5) le cas échéant la page.</p>
<p>Le menu "Inspecteur" permet d'observer le code HTML de la page web.</p>
