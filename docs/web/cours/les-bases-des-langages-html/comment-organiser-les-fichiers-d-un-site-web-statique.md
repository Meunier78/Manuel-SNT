# Comment organiser les fichiers d'un site web statique ?

<div class="imported-content">

<p><img class="imported-image ptSynthesisImg" data-filename="blob" data-height="501" data-original-url="media%2For%2F551774383%3Fhid%3Dm_16698" data-width="750" id="m_16698" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-cours-les-bases-des-langages-html-comment-organiser-les-fichiers-d-un-site-web-statique-html/image-01.jpg"/></p>
<p>Lors de la création d’un site contenant plusieurs pages web dont la page d’accueil est "index.html", il est conseillé de structurer les fichiers et les dossiers selon l’exemple ci-dessus.</p>
<p>Le dossier accueillant l'intégralité du site web peut être déposé sur un ordinateur pour une consultation locale ou à la racine d'un serveur web pour une consultation en ligne. Dans ce dernier cas, le transfert des fichiers se fait en FTP à l'aide d'un logiciel open source et gratuit tel que <a href="https://filezilla-project.org/" rel="noopener" target="_blank">FileZilla</a>.</p>
<p><b>Attention :</b> il faut impérativement écrire les noms de tous les fichiers et de tous les dossiers en minuscules, sans espace et sans accentuation.</p>
