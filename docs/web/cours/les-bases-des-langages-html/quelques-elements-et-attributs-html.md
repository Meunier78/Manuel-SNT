# Quelques éléments et attributs HTML

<p>Un élément HTML peut être soit constitué d’une paire de balises et d’un contenu, soit (plus rarement) d’une balise unique qu’on dit alors orpheline (par exemple la balise <b>br </b>qui indique un saut de ligne).</p>
<p>La balise ouvrante d’un élément HTML peut contenir des <b>attributs</b> (parfois obligatoires) possédant une valeur. Par exemple, l’élément <b>a</b> (pour « anchor ») servant à créer des liens vers d’autres sites ou d’autres pages, va avoir besoin d’un attribut <b>href</b> (« hypertext reference ») qui va prendre comme valeur l’adresse (relative ou absolue) de la page vers laquelle on souhaite faire un lien.</p>
<p>L’élément <b>img</b>, servant à insérer une image dans une page HTML, va quant à lui nécessiter deux attributs : <b>src</b> et <b>alt</b>. L’attribut <b>src</b> va prendre comme valeur le nom et l’emplacement de l’image tandis que l’attribut <b>alt</b> va afficher un texte alternatif dans le cas où l’image ne serait pas disponible.<br/></p>
