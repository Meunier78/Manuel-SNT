# Qu'est-ce que le HTML ?

<div class="imported-content">

<p><br/></p>
<p>HTML (Hypertext Markup Language) est le langage de balisage standard utilisé pour créer la structure et le contenu d'une page web. Le HTML indique la signification sémantique du contenu, c'est-à-dire la fonction et la structure des éléments sur la page. Il établit la base de la structure de la page, mais il ne se concentre pas sur l'apparence visuelle. <br/></p>
<p>Le HTML est composé de balises qui décrivent la signification et la structure des éléments sur la page. Les balises HTML sont entourées de chevrons ("&lt;" et "&gt;") et sont souvent utilisées par paires, avec une balise d'ouverture et une balise de fermeture.<br/></p>
<p>Une page web est un simple fichier texte contenant des balises et enregistré avec l’extension ".html". On peut donc construire une page web à l’aide d’un éditeur de texte gratuit tel que<b> </b><a href="https://notepad-plus-plus.org/" rel="noopener" target="_blank"><b>NotePad++</b></a> par exemple. Un fichier HTML est un simple fichier texte ayant l'extension ".html" dont la structure de base est rappelée au début de ce document. </p>
<p><b>Remarques :</b></p>
<ul><li>L’indentation et les sauts de ligne dans le code source ne sont absolument pas obligatoires (le navigateur les supprime à la lecture du fichier) : on pourrait très bien décrire une très longue page HTML dans un fichier sur une seule (très longue) ligne ! Cependant, le code source apparaît beaucoup plus clair et lisible si on prend soin de le structurer avec des indentations...<br/></li><li>Toute balise ouverte doit être refermée.<br/></li><li>Les balises doivent être correctement imbriquées : une balise incluse dans une autre doit être refermée avant que la précédente ne le soit.</li><li>On peut ajouter des commentaires dans un code HTML de la manière suivante :<br/><b>&lt; ! -- Ceci est un commentaire --&gt;</b><br/></li></ul><p><br/></p>
