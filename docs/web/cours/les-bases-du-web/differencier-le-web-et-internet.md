# Différencier le Web et Internet

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>Internet est un réseau mondial connectant des millions d'ordinateurs et d'appareils à travers le monde. Il s'agit de l'infrastructure physique composée de câbles, de routeurs, de serveurs et d'autres périphériques réseau qui permettent l'échange de données entre les périphériques connectés. Internet vous permet de transférer toutes sortes d'informations telles que des fichiers, des e-mails, des vidéos, des images, etc.<br/><br/>Le Web, quant à lui, est un système d'information qui fonctionne sur Internet. En anglais, on l'appelle souvent le "Web" ou "World Wide Web". Le Web utilise des protocoles spécifiques, tels que HTTP (Hypertext Transfer Protocol), pour organiser et présenter les informations sur Internet de manière conviviale. Le Web est composé de pages web qui contiennent du texte, des images, des hyperliens, des vidéos et d'autres ressources. Les sites web sont reliés entre eux par des hyperliens, ce qui facilite la navigation d'une page à l'autre.<br/><br/>En résumé, Internet est l'infrastructure physique qui permet les connexions entre les ordinateurs et les appareils, et le Web est un système d'information qui utilise cette infrastructure pour afficher et organiser les données de manière interactive. <br/></p>
