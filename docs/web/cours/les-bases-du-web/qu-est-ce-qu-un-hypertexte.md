# Qu'est-ce qu'un hypertexte ?

<div class="imported-content">

<div class="card">




<div id="note-text"><p><img class="ptSynthesisImg imported-image" data-filename="LE WEB 01.png" data-height="590" data-original-url="media%2For%2F551774345%3Fhid%3Dm_16574" data-width="750" id="m_16574" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-cours-les-bases-du-web-qu-est-ce-qu-un-hypertexte-html/image-01.jpg"/></p>
<p>L'hypertexte est un concept fondamental du World Wide Web (www).<br/></p>
<p>Un hypertexte permet de créer des liens interactifs entre différentes parties d'un document ou entre des documents distincts. En d'autres termes, l'hypertexte est une technologie qui utilise des hyperliens pour associer un contenu textuel à d'autres informations connexes.<br/><br/>Un hypertexte ou un lien hypertexte est un élément cliquable dans un document (tel qu'une page web ou un document électronique) qui, lorsqu'il est activé, envoie automatiquement vers un autre emplacement dans le même document ou vers un autre document. En cliquant sur un lien hypertexte, vous accédez à une autre page web, à un fichier, à une image, à une vidéo ou à une autre ressource connexe.<br/></p>
<p>L'hypertexte permet une navigation non linéaire entre les informations. Au lieu de lire un document du début à la fin, les utilisateurs peuvent cliquer sur des liens pour accéder aux informations pertinentes, ce qui facilite la recherche d'informations spécifiques.<br/></p>
<p>L'hypertexte a grandement contribué à l'essor d'Internet en tant que source d'information interactive.<br/></p>
