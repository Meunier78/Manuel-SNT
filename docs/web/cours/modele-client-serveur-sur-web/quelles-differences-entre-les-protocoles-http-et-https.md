# Quelles différences entre les protocoles HTTP et HTTPS ?

<div class="imported-content">

<div class="card">




<div id="note-text"><p><img class="imported-image ptSynthesisImg" data-filename="blob" data-height="653" data-original-url="media%2For%2F551774367%3Fhid%3Dm_16700" data-width="750" id="m_16700" loading="lazy" src="/Manuel-SNT/assets/imported/le-web-cours-modele-client-serveur-sur-web-quelles-differences-entre-les-protocoles-http-et-https-html/image-01.jpg"/></p>
<p>Le protocole HTTPS est une surcouche du protocole HTTP qui permet de sécuriser une connexion HTTP par chiffrement des données et authentification du serveur à l'aide d'un certificat. Il est devenu le standard des échanges de communication entre un navigateur et un serveur web. Dans le navigateur, il se manifeste par un cadenas qui apparaît devant l'URL de la page web visitée.<br/></p>
