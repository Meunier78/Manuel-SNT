# La responsabilité des usagers

<div class="imported-content">

<p>En tant qu'internaute, vous devez être conscient de vos responsabilités. Ne publiez pas de contenu diffamatoire, abusif ou illégal. Respectez les droits des autres et évitez d'une manière générale de nuire à autrui en ligne. <br/></p>
