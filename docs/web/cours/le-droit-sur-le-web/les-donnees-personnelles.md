# Les données personnelles

<div class="imported-content">

<p>L'une des principales préoccupations du droit du Web concerne la protection des données personnelles. Les utilisateurs doivent être conscients de leurs droits concernant la collecte, le traitement et le stockage des données par les entreprises en ligne. Des réglementations telles que le Règlement général sur la protection des données (RGPD) de l'Union européenne sont en place pour garantir la confidentialité et la sécurité des données personnelles. <br/></p>
<p>Ce cadre protecteur conduit certains internautes a revendiquer de plus en plus leur droit à la vie privée sur le Web, exigeant que certaines données (photos, coordonnées, etc.) ne soient pas rendues publiques. Alors que la <a href="https://www.cnil.fr/fr" rel="noopener" target="_blank">CNIL</a> (Commission Nationale de l'Informatique et des Libertés) a documenté de nombreuses plaintes concernant la diffusion de données personnelles sans consentement, le gouvernement parle d'un « droit numérique à l'oubli » : tous les citoyens ont, par exemple, le droit de faire retirer leurs informations d'un site web sur simple demande. <br/></p>
