# 2001 - Standardisation des pages grâce au DOM (Document Object Model)

<div class="card">





<div><a class="relative-url" href="https://commons.wikimedia.org/wiki/File:DOM-model.svg" rel="noopener" target="_blank"><p class="wrapped-paragraph"><img class="wrapped ptSynthesisImg imported-image" data-original-url="media%2FpearlContent%2F551774276" id="l_2420680" loading="lazy" pearltrees="true" src="/Manuel-SNT/assets/imported/le-web-reperes-2001-standardisation-des-pages-grace-document-object-model-html/image-01.jpg" width="428"/></p></a></div><div id="note-text"><p>Le DOM (Document Object Model) est une représentation hiérarchique d'un document HTML ou XML, qui permet aux programmes informatiques d'accéder et de manipuler les éléments d'une page web de manière dynamique. Il fournit une interface de programmation standardisée pour interagir avec les éléments d'une page, tels que la modification du contenu, la manipulation des styles ou la gestion des événements.<br/></p>
