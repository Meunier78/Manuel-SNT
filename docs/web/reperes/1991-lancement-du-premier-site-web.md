# 1991 - Lancement du premier site web

<div class="imported-content">

<p>En 1991, Tim Berners-Lee met en ligne le premier site web public. Il est composé de quelques pages web, codées à la main en HTML, qui présentent des informations sur le World Wide Web lui-même, offrant un aperçu de son potentiel. Une copie de son dernier état, datant de 1993, est consultable sur le site du CERN. Ce premier site web public était hébergé sur un ordinateur NeXT qui est ainsi devenu le premier serveur web au monde.<br/></p>
