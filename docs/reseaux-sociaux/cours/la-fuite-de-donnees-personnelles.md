# La fuite de données personnelles

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Il est important de considérer avec soin son authentification sur les réseaux sociaux : si quelqu'un connait votre identifiant et votre mot de passe, il peut se connecter sur un réseau à votre place et en faire un usage qui engage votre responsabilité.  <br/></p>
<p>Une authentification à deux facteurs est une protection supplémentaire : outre le mot de passe, elle oblige à disposer du téléphone portable de la personne, sur lequel un SMS avec un code est envoyé pour valider l'authentification.<br/></p>
