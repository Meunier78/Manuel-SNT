# Comment différencier les réseaux sociaux ?

<div class="imported-content">

<p>Les réseaux sociaux peuvent se ressembler, mais la plupart ont des spécificités. Qu'est-ce qui rend chaque réseau social unique ? <br/></p>
<p>On peut différencier les réseaux sociaux en fonction :</p>
<ul><li>de leurs fonctionnalités (ce qu'ils permettent de faire, notamment les modalités de partage des contenus, ou d'interaction avec les autres utilisateurs) ;<br/></li><li>du type ou du format des contenus partagés (photos, vidéos, textes, liens, messages vocaux) ;<br/></li><li>de leur modèle économique (publicité, abonnement, services payants). </li></ul><p>Dans le cas de Linkedin, le public cible est spécifique : il s'agit d'adultes en activité professionnelle ou en recherche d'emploi. Pour les autres réseaux, le public peut évoluer en fonction du marketing ou des modes. Il n'y a pas si longtemps, Facebook était également beaucoup utilisé par les "jeunes" !<br/><br/></p>
