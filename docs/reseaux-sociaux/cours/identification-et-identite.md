# Identification et identité

<div class="imported-content">

<p>Pour interagir sur les réseaux sociaux, il faut créer un compte qui permettra de s'identifier. Sur certains réseaux, on est invité à se présenter avec son vrai nom. Sur d'autres, les pseudos sont possibles voire recommandés. <br/></p>
<p>Mais lorsque l'on crée un profil avec un pseudo sans faire apparaître son véritable nom, il peut y avoir un lien avec celui-ci. Par exemple, si je crée un compte Instagram avec un pseudo, j'indique une adresse électronique. Si c'est celle de mon compte nominatif Google, il devient possible d'établir un lien avec ma véritable identité. <br/></p>
<p>Ainsi, même si je n'indique pas mon vrai nom sur un réseau, mon identité peut être retrouvée en croisant des informations. L'anonymat parfait n'existe pas. <br/></p>
