<p class="scrap-without-first-image"></p><div id="note-text"><h2>Excentricité d'un sommet du graphe<br/></h2><p>L’excentricité d’un sommet est la distance maximale entre ce sommet et les autres sommets du graphe. Ainsi, l'excentricité de Luc est de 3 (distance avec Inès ou Louna). L'excentricité de Gabriel est de 4 (distance avec Louna).</p>
<p>Voici la liste des excentricités de tous les membres du réseau.</p>
<ul><li>Alexandre : 3</li><li>Emma : 3</li><li>Gabriel : 4</li><li>Hugo : 4</li><li>Inès : 4</li><li>Leila : 4</li><li>Louna : 4</li><li>Luc : 3</li><li>Margot : 4<br/></li></ul><h2>Centre(s) du graphe</h2><p>Le centre d’un graphe est le sommet dont l’excentricité est la plus petite. Il peut y avoir plusieurs centres dans un graphe.<br/></p>
<h2>Rayon d’un graphe<br/></h2><p>Le rayon d’un graphe, c’est l’excentricité du centre du graphe (ou de l’un des centres s’il y en a plusieurs).<br/></p>
<p>Le rayon du graphe est ici égal à 3.<br/></p>
<h2>Diamètre d'un graphe </h2><p>Le diamètre d’un graphe est la distance maximale entre deux sommets de ce graphe.<br/><br/></p>
