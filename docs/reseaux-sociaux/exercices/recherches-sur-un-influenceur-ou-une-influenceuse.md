# Recherches sur un influenceur ou une influenceuse

<div class="imported-content">

<p>1. Choisissez un influenceur ou une influenceuse. Présentez cette personne en quelques lignes : quelle est son identité, quel a été son parcours ? Quels sont les sujets qu'elle traite ? <br/></p>
<p>2. Cette personne vit-elle de son travail d'influenceur/influenceuse ? D'après vous, quelle est sa source de revenus ? Précisez les sources d'information qui vous permettent de le penser. <br/></p>
