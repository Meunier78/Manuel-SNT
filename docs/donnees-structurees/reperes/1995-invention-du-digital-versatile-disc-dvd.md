# 1995 - Invention du Digital Versatile Disc (DVD)

<div class="imported-content">

<p>Le Digital Versatile Disc (DVD) permet de stocker les données vidéo sur un disque optique sous la forme d’un signal numérique au moyen d’un code binaire.<br/></p>
