# 2006 - Commercialisation du blu-ray disc (BD)

<div class="imported-content">

<p>Le blu-ray disc (BD) permet de stocker les données vidéo en haute définition sur un disque optique sous la forme d’un signal numérique au moyen d’un code binaire.<br/></p>
