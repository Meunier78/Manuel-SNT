# 1991 - Premier Solid State Drive (SSD)

<div class="imported-content">

<p>Le Solid State Drive (SSD) est dépourvu de pièce mécanique en mouvement. Il permet le stockage de données sur de la mémoire flash avec un temps d'accès rapide.<br/></p>
