# 1970 - Invention du modèle relationnel

<div class="imported-content">

<p>Le modèle relationnel est une manière de modéliser les relations existantes entre plusieurs informations, et de les ordonner entre elles. Cette modélisation qui repose sur des principes mathématiques inventé par E.F. Codd est souvent implémentée physiquement dans une base de données.<br/></p>
