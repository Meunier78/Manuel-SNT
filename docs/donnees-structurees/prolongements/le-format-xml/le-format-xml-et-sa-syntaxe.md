# Le format XML et sa syntaxe

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Le XML (eXtensible Markup Language) est un langage de balisage utilisé pour stocker et transporter des données. Le XML est un format de données très flexible et largement utilisé dans les applications web, les applications mobiles, les bases de données, les systèmes d'échange de données, et bien d'autres domaines. Voici ci-dessous quelques concepts de base du XML.</p>
<ul><li><b>Balises</b><br/>Les balises sont les éléments de base du XML. Elles sont définies par des chevrons (&lt; et &gt;) et sont utilisées pour encapsuler les données. Les balises sont souvent utilisées par paires, avec une balise d'ouverture et une balise de fermeture.</li><li><b>Attributs</b><br/>Les attributs fournissent des informations supplémentaires sur une balise. Ils sont définis à l'intérieur de la balise d'ouverture et sont séparés du nom de la balise par un espace.</li><li><b>Éléments </b><br/>Les éléments sont les éléments de base d'un document XML. Ils peuvent contenir des balises, des attributs et/ou du texte. Les éléments peuvent également contenir d'autres éléments, ce qui permet de créer des structures de données hiérarchiques.</li><li><b>Document</b><br/>Un document XML est un fichier texte qui contient des balises, des attributs et du texte. Un document XML commence souvent par une déclaration XML, qui spécifie la version du XML utilisée et l'encodage des caractères du document.</li></ul></div>
