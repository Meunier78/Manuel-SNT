# Qu'est-ce qu'un objet ?

<div class="imported-content">

<div class="card">




<div id="note-text"><p><img class="imported-image ptSynthesisImg" data-filename="blob" data-height="764" data-original-url="media%2For%2F551775272%3Fhid%3Dm_16738" data-width="750" id="m_16738" loading="lazy" src="/Manuel-SNT/assets/imported/donnees-structurees-traitement-cours-qu-est-ce-qu-un-objet-html/image-01.jpg"/></p>
<p><br/></p>
<p>En informatique un objet permet de modéliser des entités ou des concepts du monde réel dans un programme. </p>
<p>Par exemple un contact dans un carnet d’adresses peut-être modélisé sous la forme d’un objet regroupant plusieurs données associées à  des descripteurs.</p>
<p>Une collection regroupe des objets possédant les mêmes descripteurs.</p>
