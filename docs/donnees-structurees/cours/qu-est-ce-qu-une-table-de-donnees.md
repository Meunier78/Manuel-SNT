# Qu'est-ce qu'une table de données ?

<div class="imported-content">

<div class="card">




<div id="note-text"><p><img class="imported-image ptSynthesisImg" data-filename="blob" data-height="258" data-original-url="media%2For%2F551775271%3Fhid%3Dm_16740" data-width="750" id="m_16740" loading="lazy" src="/Manuel-SNT/assets/imported/donnees-structurees-traitement-cours-qu-est-ce-qu-une-table-de-donnees-html/image-01.jpg"/></p>
<p>Les données structurées peuvent être organisées sous la forme de tableaux à deux dimensions appelés tables de données. Une table de données permet de stocker de façon persistante une collection d’objets dans un fichier ou une base de données.<br/></p>
<p>Les tables de données sont organisées en lignes et en colonnes. À l’intersection des deux, on trouve une donnée. En particulier :<br/></p>
<ul><li>la première ligne du tableau correspond aux descripteurs permettant de caractériser les données ;<br/></li><li>chaque ligne suivante correspond à un objet qui contient la valeur des différents descripteurs ;<br/></li><li>chaque colonne correspond à un descripteur commun à tous les objets de la collection.</li></ul><p> Un tableur tel que Calc de la suite bureautique libre et gratuite <a href="https://fr.libreoffice.org/" rel="noopener" target="_blank">LibreOffice</a> permet le traitement des tables de données. Dans l’exemple au début de cette page, chaque élève est un objet caractérisé par huit valeurs de données associées aux huit descripteurs (Nom, Prénom, Sexe, Date de naissance, Classe, Demi-pension, LV1, LV2) de la collection.</p>
<p>Les tables de données sont couramment utilisées dans les applications de gestion de bases de données pour stocker et gérer de grandes quantités d’informations structurées, telles que des données de clients, des informations de produits, des enregistrements de ventes, des données de compte, etc. </p>
<p>Les tables de données permettent de rechercher et trier facilement les informations en fonction de différents critères. Elles permettent aussi de les filtrer, les analyser et les manipuler à l’aide de requêtes et de fonctions spécifiques.</p>
