# Qu'est-ce qu'une information ?

<div class="imported-content">

<p class="scrap-without-first-image"></p><p>Une information est une donnée qui a été traitée et interprétée pour en extraire une signification pour les utilisateurs dans un contexte particulier.<br/></p>
<p><b>Exemples de données :</b></p>
<ul><li>une date : 19 mars 2023 ;<br/></li><li>un nombre de visiteurs : 500 ;<br/></li><li>un code postal : 75007 ;<br/></li><li>une taille : 1,75 m ;<br/></li><li>une vitesse : 100 km/h.</li></ul><p><b>Exemples d'informations :</b></p>
<ul><li>la date est le jour de l’équinoxe de printemps ;<br/></li><li>le nombre de visiteurs est en hausse de 20 % par rapport à la même période de l’année dernière ;<br/></li><li>le code postal correspond à l’arrondissement de la tour Eiffel à Paris ;<br/></li><li>la taille correspond à la moyenne pour un homme adulte en France ;<br/></li><li>la vitesse est supérieure à la limite de vitesse autorisée sur cette route.</li></ul><p><img ="xl"="" class="imported-image" data-height="666" data-original-url="logo%2Fhash%2F551775259%3Fh%3D8d89e6dd65f89d8c50f85db6a8f58171" data-width="1000" id="l_2381803" loading="lazy" src="/Manuel-SNT/assets/imported/donnees-structurees-traitement-cours-qu-est-ce-qu-une-information-html/image-01.jpg"/></p>
