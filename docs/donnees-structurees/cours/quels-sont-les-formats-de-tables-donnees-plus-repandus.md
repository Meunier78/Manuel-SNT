# Quels sont les formats de tables de données les plus répandus ?

<div class="imported-content">

<p>Les formats XLS, XLSX et ODS sont les plus utilisés par le grand public. Ils permettent d’afficher une table de données sous la forme conventionnelle d’un tableau à deux dimensions. Il est d’usage de faire correspondre à chaque objet un identifiant unique noté « id » afin de faciliter les traitements ultérieurs.</p>
<p>Remarque : afin d’éviter les erreurs d’encodage de caractères lors de la conversion d’un format de table de données en un autre, il est recommandé de ne pas accentuer les caractères latins des noms des descripteurs.<br/></p>
