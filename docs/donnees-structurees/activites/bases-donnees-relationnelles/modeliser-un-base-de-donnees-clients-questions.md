<p class="scrap-without-first-image"></p><div id="note-text"><p>Téléchargez sur votre ordinateur les fichiers intitulés fichier_clients_maquette1.xlsx et fichier_clients_maquette2.xlsx. À l’aide du tableur Excel de Microsoft Office ou de <a href="https://fr.libreoffice.org/download/telecharger-libreoffice/" rel="noopener" target="_blank">Calc</a> de la suite bureautique libre et gratuite <a href="https://fr.libreoffice.org/" rel="noopener" target="_blank">LibreOffice</a>, ouvrez ces fichiers et répondez aux questions suivantes. <br/></p>
<p>1. En termes d'optimisation d'enregistrement des données, quel est le principal inconvénient que présente la structure de <i>fichier_clients_maquette1 </i>?<br/></p>
<p>2. Afin d'éviter cet inconvénient, Jeanne décide dans <i>fichier_clients_maquette2 </i>de séparer les données précédentes en 4 tables.</p>
<p>a)<b> </b>Quelles sont les 4 tables de données créées par Jeanne ?</p>
<p>b) Précisez pour chacune d'entre elles les descripteurs.</p>
<p>c) Une clé primaire est un descripteur permettant de définir d'une manière unique une table de données. Une clé étrangère est un descripteur permettant de relier 2 tables de données entre elles. Dans chaque table de données, identifiez la clé primaire et la (ou les) clés étrangère(s).<br/></p>
<p>d) Représentez graphiquement les tables de données et les liens qui les unissent. Les clés primaires seront en gras et soulignées et les clés étrangères en gras. L'ensemble constitue la base de données relationnelle.</p>
