# Les bases de données relationnelles

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>Jeanne, administratrice indépendante de base de données, est embauchée par une boutique de vente en ligne de matériel informatique, qui vient d'ouvrir. Elle est chargée de concevoir la base de données des clients. Elle doit pour cela réaliser une maquette qu'elle devra présenter pour validation au responsable de la boutique. Ce dernier lui a transmis un fichier au format ODS regroupant les données devant figurer dans la base.<br/></p>
