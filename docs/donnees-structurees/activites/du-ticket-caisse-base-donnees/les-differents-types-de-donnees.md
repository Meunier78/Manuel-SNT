# Les différents types de données

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Pour optimiser la gestion de l’espace de stockage disque et faciliter l’organisation des enregistrements, chaque donnée est d’un type précis. Les types de données les plus fréquents sont :</p>
<ul><li>numérique entier ou réel ;</li><li>texte (taille illimitée) ou chaîne de n caractères ;</li><li>date (jj/mm/aaaa), heure (hh:mm) ;</li><li>booléen (uniquement 2 valeurs possibles : vrai/oui ou faux/non).</li></ul></div>
