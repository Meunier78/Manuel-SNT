# Croisement des données - Questions

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Le but premier apparent de ce système informatique, adossé aux tickets de caisse, est d’améliorer la gestion comptable, logistique et commerciale du magasin. Cependant, la masse de données collectées et les traitements  associés permettent d'effectuer des prédictions sur les habitudes de consommation des clients et leurs vies.</p>
<p>1. Quel client semble faire lui-même l'entretien de sa voiture ?</p>
<p>2. Quel client a peut-être des problèmes d'équilibre alimentaire ?</p>
<p>3. Quel type d'exploitation le magasin peut-il faire des données collectées ?</p>
<p>4. Si ces données fuitent suite à un piratage, quelles peuvent être les conséquences pour les clients ?</p>
