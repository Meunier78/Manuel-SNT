# Vélib à Paris - Premières questions

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Le jeu de données à étudier est accessible depuis la page web <a href="https://opendata.paris.fr/explore/dataset/velib-disponibilite-en-temps-reel/information/?disjunctive.name&amp;disjunctive.is_installed&amp;disjunctive.is_renting&amp;disjunctive.is_returning&amp;disjunctive.nom_arrondissement_communes" rel="noopener" target="_blank">Vélib – Vélos et bornes – Disponibilités en temps réel</a> dans l'onglet "Export".<br/></p>
<p>Téléchargez le jeu de données au format Excel. À partir du fichier XLSX téléchargé, répondez aux questions suivantes, après l'avoir ouvert avec un tableur tel que Calc de la suite bureautique LibreOffice.</p>
<p>1. Les données sont-elles structurées ? Justifiez.</p>
<p>2. Sont-elles régulièrement actualisées ?</p>
<p>3. Sous quel type de licence sont-elles distribuées ?</p>
<p>4. Les données sont-elles personnelles ?</p>
<p>5. Combien le jeu de données (data set) contient-il de descripteurs ?</p>
<p>6. Combien le jeu de données (data set) contient-il d'objets (ou d'enregistrements) ?</p>
