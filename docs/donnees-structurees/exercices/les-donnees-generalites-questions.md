<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>1. Quelle est la différence entre une donnée et une information ? Illustrez cela avec au moins un exemple.<br/></p>
<p>2. Afin de respecter le RGPD, quelles sont les obligations auxquelles sont soumises les organisations qui hébergent des données personnelles d'utilisateurs ?</p>
<p>3. Pour un utilisateur, quels sont les droits garantis par le RGPD ?</p>
<p>4. Expliquez les risques que présente le croisement des données personnelles d’un utilisateur.</p>
<p>5. À partir d'un exemple de données structurées, illustrez la notion d'objet et de collection.<br/></p>
<p>6. Comment est structurée une table de données ?</p>
<p>7. Citez les formats de tables de données les plus courants.<br/></p>
<p>8. Citez au moins trois types de stockage local de données.</p>
<p>9. Citez quelques avantages et inconvénients du stockage des données dans le cloud.<br/></p>
<p>10. Citez quelques avantages et inconvénients du big data.</p>
