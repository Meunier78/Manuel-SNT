# Cinémas actifs en France - Questions avec le tableur

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Dès lors que la recherche comporte plusieurs critères devant être vérifiés simultanément, il devient nécessaire de travailler localement sur le jeu de données afin d'utiliser des fonctionnalités avancées (tri et filtrage) d'un tableur.<br/></p>
<p>Téléchargez le jeu de données au format Excel. On pourra utiliser pour l'exploitation le tableur <a href="https://fr.libreoffice.org/" rel="noopener" target="_blank">Calc de la suite bureautique libre et gratuite LibreOffice</a>.</p>
<p>1. En combinant deux filtrages dans le tableur, trouvez le cinéma en Auvergne / Rhône-Alpes ayant programmé le plus grand nombre de films.</p>
<p>2. En combinant un filtrage et un tri dans le tableur, afficher les cinémas de Lyon par ordre décroissant d'entrées en 2021.</p>
