# Métadonnées d'une photo - Définitions

<div class="imported-content">

<p class="scrap-without-first-image"></p><p>Un fichier image issu d'un appareil photo numérique ou d'un smartphone contient plus que les données relatives au codage de l'image elle même... En effet, les fichiers image contiennent généralement des données EXIF (Exchangeable Image File).<i> </i>Les EXIF sont un ensemble de données relatives à chaque photo et présentées dans un format normalisé. Ces données sont générées automatiquement par l’appareil photo lors de la prise de vue. Elles sont stockées dans le fichier image pour les formats JPG, TIFF et RAW.<br/></p>
