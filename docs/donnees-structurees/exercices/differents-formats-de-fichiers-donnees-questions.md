# Différents formats de fichiers de données - Questions

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>En vous servant du site <a href="https://codebeautify.org/" rel="noopener" target="_blank">Code Beautify</a> qui permet de convertir un format de fichier en un autre, répondez aux questions suivants. Cette <a href="https://codebeautify.org/csv-to-xml-json" rel="noopener" target="_blank">page</a> est plus particulièrement dédiée aux conversions de CSV en XML ou JSON.<br/></p>
<p>1. Copiez puis collez le code du fichier CSV contacts_perso.csv et convertissez le en XML.</p>
<p>a) Quelles sont les 3 balises qui constituent la structure de base d'un fichier XML ?</p>
<p>b) Comment code-t-on un objet en XML ? </p>
<p>c) Comment sont séparés des objets en XML ?</p>
<p><br/></p>
<p>2. Copiez puis collez le code du fichier CSV contacts_perso.csv et convertissez le en JSON.</p>
<p>a) Par quel caractère commence un fichier JSON et par lequel finit-il ?</p>
<p>b) Comment code-t-on un objet en JSON ?</p>
<p>c) Comment sont séparés des objets en JSON ?</p>
<p><br/></p>
<p>3. Les mentions légales du site <a href="https://codebeautify.org/" rel="noopener" target="_blank">Code Beautify</a> précisent-elles où le site est hébergé ? Si tel n'est pas le cas utilisez le site <a href="https://www.whois.com/whois" rel="noopener" target="_blank">https://www.whois.com</a> pour le savoir.</p>
<p>4. On trouve sur Internet de nombreux sites permettant de convertir en ligne des fichiers de données d'un format dans un autre. Selon vous, cela pose-t-il problème ?</p>
