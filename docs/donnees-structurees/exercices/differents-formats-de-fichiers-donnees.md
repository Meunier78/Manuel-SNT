# Différents formats de fichiers de données

<div class="imported-content">

<p>L'un des avantages du format CSV est sa grande portabilité : il peut par exemple être facilement converti dans d'autres formats tels que XML et JSON.</p>
<p>XML est un sigle qui est l'abréviation de <i>eXtensible Markup Language, </i>format basé sur l'utilisation de balises pour structurer les données. Les balises sont utilisées pour encadrer un contenu : il y a une balise ouvrante  et une balise fermante. Le format d'un fichier XML est facilement consultable par un humain et compréhensible.</p>
<p>Le format <i>JavaScript Object Notation</i> (JSON) est un format utilisé pour représenter des objets qui dérive de la notation des objets du langage JavaScript. Un document JSON est un ensemble de paires constituées d'une étiquette et d'une valeur ou d'une liste de valeurs. Les paires sont placées à l'intérieur d'accolades et séparées par des virgules. Les valeurs des listes sont quant à elles placées entre crochets et séparées par des virgules.<br/></p>
