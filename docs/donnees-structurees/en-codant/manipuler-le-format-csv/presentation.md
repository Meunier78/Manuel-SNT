<p class="scrap-without-first-image"></p><div id="note-text"><p>On trouve énormément de données sur Internet. Une partie de ces données sont publiques.</p>
<p>Par exemple le site <a href="http://www.data.gouv.fr/" rel="noopener" target="_blank">data.gouv.fr</a> recense un grand nombre de données librement réutilisables. Ces données sont souvent contenues dans un fichier au format CSV, ce qui est très courant sur Internet.<br/></p>
<p>Cette activité va vous permettre d'extraire des informations utiles d'un tel fichier.<br/></p>
