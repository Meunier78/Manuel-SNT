<p class="scrap-without-first-image"></p><div id="note-text"><p>JavaScript Object Notation (JSON) est un format de données textuelles dérivé de la notation des objets du langage JavaScript. Il permet de représenter de l’information structurée. Il a été créé par l’américain Douglas Crockford qui l'a décrit en 2006.<br/><br/><b>Description :</b><br/><i>{...}</i> : les accolades définissent un objet.<br/><i>"language":"Java"</i> : Les guillemets (double-quotes) et les double-points définissent un couple clé/valeur (on parle de membre).<br/><i>[...]</i> : Les crochets définissent un tableau (ou array en anglais).<br/><i>{"id":1, "language":"json", "author":"Douglas Crockford"}</i> : Les virgules permettent de séparer les membres d'un tableau ou, comme ici, d'un objet.<br/></p>
