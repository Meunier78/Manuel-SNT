# Découvrir le langage machine - Questions

<div class="imported-content">

<p>1. Qu'est-ce qu'un octet ?<br/></p>
<p>2. Combien un octet peut-il stocker d'informations ?</p>
<p>3. Convertissez le nombre décimal 67 en binaire puis codez le sur 1 octet.<br/></p>
<p>4. Comment code-t-on le nombre 67 en hexadécimal ?</p>
<p>5. Qu'est-ce que le code ASCII ? Quel est son principal inconvénient ?</p>
<p>6. Donnez le code ASCII décimal puis hexadécimal de la lettre minuscule z. Comment est-elle codée sur 1 octet en binaire ?</p>
<p>7. Même question pour la lettre majuscule Z.</p>
<p>8. Une séquence de 5 caractères est tapée sur le clavier d'un ordinateur. Cette séquence en binaire est la suivante : 01010011 01001110 01010100 00100000 00100001<br/></p>
<p>Quelle est l'information codée ?</p>
<p>9. Quel est le code ASCII hexadécimal de la séquence composée des quatre caractères suivants : 2nde ?<br/></p>
<p>10. Comment l'information précédente est-elle codée en binaire sous la forme de quatre octets ?<br/></p>
