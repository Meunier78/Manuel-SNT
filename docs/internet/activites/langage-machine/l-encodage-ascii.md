# L'encodage ASCII

<div class="imported-content">

<p><img ="xl"="" class="imported-image ptSynthesisImg" data-height="499" data-original-url="logo%2Fhash%2F551774000%3Fh%3D0dcfde715474283ac41a2f6ea950ec24" data-width="750" id="l_2386073" loading="lazy" src="/Manuel-SNT/assets/imported/internet-activites-langage-machine-l-encodage-ascii-html/image-01.jpg"/></p>
<p>Le codage des caractères non accentués repose sur la norme ASCII (American Standard Code for Information Interchange) . L'ASCII définit 128 caractères numérotés de 0 à 127 et codés en binaire sur 8 bits (1 octet) de 00000000 à 01111111.<br/></p>
