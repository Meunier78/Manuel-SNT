<p>To navigate, press the arrow keys.</p>
<p>Linux: traceroute -I -q1 example.com | xclip -sel clip</p>
<p>Mac: traceroute -I -q1 example.com | pbcopy</p>
<p>Windows: tracert example.com | clip</p>
