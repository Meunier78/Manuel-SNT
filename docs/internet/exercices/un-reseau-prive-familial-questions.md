# Un réseau privé familial - Questions

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>On considère une machine sur un réseau privé familial ayant les caractéristiques suivantes :<br/></p>
<p>  IP             /     Masque<br/>192.168.100.9 / 255.255.255.0<br/></p>
<p>1. Quelle est l'adresse IPv4 du réseau ?<br/></p>
<p>2. Quelles sont les adresses IPv4 autorisées sur ce réseau ?<br/></p>
<p>3. Quelle est l'IPv4 de broadcast de ce réseau ? À quoi sert cette adresse IPv4 particulière ?</p>
<p>4. Combien de machines ce réseau peut-il contenir ?</p>
<p>5. Dans la forme abrégée 192.168.100.9 / x, quelle est la valeur de x ?</p>
