# Stockage et transferts d'informations - Questions

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>1. Un disque dur classique a une capacité de stockage de 1 To. Combien de Go cela représente-t-il ?</p>
<p>2. On considère un roman composé de 300 pages. Sachant qu'il a été rédigé avec un traitement de texte pouvant mettre environ 4512 caractères (espaces compris) au maximum sur une page, combien de romans de 300 pages pourrait-on stocker sur un disque dur classique, sachant qu'un caractère est codé sur un octet ?</p>
<p>3. En France, le débit moyen descendant (download) d'une box raccordée en fibre optique est de 500 Mb/s alors que celui d'une connexion 4G/5G est de 50 Mb/s. Combien de temps vous faudrait-il pour télécharger sur votre PC la version numérique du roman précédent depuis un site marchand via une connexion fibre ? Même question sur votre smartphone via une connexion 4G/5G.<br/></p>
<p>4. Votre clé USB de 32 Go contenant vos photos de vacances est pleine. Vous souhaitez effectuer la sauvegarde de ces photos sur un cloud. Combien de temps vous faudra-t-il pour effectuer cette sauvegarde en utilisant :</p>
<ul><li>une connexion internet ADSL dont le débit montant (upload) est de 10 Mb/s ;<br/></li><li>une connexion internet fibre optique dont le débit montant (upload) est de 200 Mb/s.</li></ul><p><img class="imported-image" data-filename="open-hard-drive-1200164_1280.jpg" data-height="654" data-original-url="media%2For%2F551774123%3Fhid%3Dm_16649" data-width="750" id="m_16649" loading="lazy" src="/Manuel-SNT/assets/imported/internet-exercices-stockage-et-transferts-d-informations-questions-html/image-01.jpg"/></p>
