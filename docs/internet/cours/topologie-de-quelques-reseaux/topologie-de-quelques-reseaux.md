<p>La topologie d'un réseau informatique correspond à son architecture physique : elle définit les connexions entre les équipements et les hiérarchies possibles. </p>
<p>Dans les réseaux grands publics ou d'entreprises on distingue deux grandes familles en fonction du mode de propagation de l'information.</p>
<p><b>Mode de diffusion (par exemple topologie en bus ou en anneau)</b><br/>Ce mode de fonctionnement consiste à utiliser un support unique de transmission. Le principe est que le message est envoyé sur le réseau : toute machine est capable de recevoir le message et d'analyser selon l'adresse du destinataire si le message lui est destiné ou non.</p>
<p><b>Mode point à point (par exemple topologie en étoile ou en maille)</b><br/>Dans ce mode, le support physique ne relie qu'une paire d'unités seulement. Pour que deux unités réseaux communiquent, elles passent obligatoirement par un intermédiaire : le nœud.<br/></p>
