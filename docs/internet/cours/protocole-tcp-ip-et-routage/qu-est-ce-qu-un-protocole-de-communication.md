# Qu'est-ce qu'un protocole de communication ?

<div class="imported-content">

<p><img class="imported-image ptSynthesisImg" data-filename="INT COURS1.png" data-height="545" data-original-url="media%2For%2F551774089%3Fhid%3Dm_16614" data-width="750" id="m_16614" loading="lazy" src="/Manuel-SNT/assets/imported/internet-cours-protocole-tcp-ip-et-routage-qu-est-ce-qu-un-protocole-de-communication-html/image-01.jpg"/></p>
<p>Lorsque deux personnes se parlent afin d'échanger des informations, elles suivent un protocole faisant intervenir, d'une manière simplifiée, trois étapes :</p>
<ul><li>établissement de la communication ;<br/></li><li>échange d'informations ;<br/></li><li>terminaison de la communication.</li></ul><p>En informatique, un protocole est un ensemble de règles qui régissent la transmission de l'information sur un réseau. Il existe de nombreux protocoles, chacun spécialisé dans une tâche bien précise : DHCP, DNS, FTP, HTTP, HTTPS, POP, SMTP, SSH...<br/></p>
<p>D'une manière générale, on peut décomposer un protocole de communication entre deux machines (ou services) en sept étapes. <br/></p>
