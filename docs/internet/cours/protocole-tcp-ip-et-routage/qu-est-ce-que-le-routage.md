# Qu'est-ce que le routage ?

<div class="imported-content">

<p>Dans les réseaux à commutation de paquets, tels qu'Internet, le processus de routage consiste à sélectionner les chemins que doivent emprunter les paquets IP (Internet Protocol) pour se rendre de leur origine à leur destination. Ces décisions de routage Internet sont prises par des périphériques réseau spécialisés qui sont appelés routeurs.<br/></p>
