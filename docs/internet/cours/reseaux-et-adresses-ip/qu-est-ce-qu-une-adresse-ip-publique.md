# Qu'est-ce qu'une adresse IP publique ?

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Les adresses IP publiques sont nécessaires pour les appareils qui doivent communiquer sur le réseau global Internet ou être accessibles depuis Internet. Les fournisseurs d'accès à Internet (FAI) attribuent généralement des adresses IP publiques aux routeurs (box) domestiques et aux entreprises, leur permettant de se connecter à Internet et d'accéder aux services en ligne.<br/></p>
<p>Les adresses IP publiques doivent être uniques à l'échelle mondiale. Elles sont attribuées par les organismes de réglementation des adresses IP, comme l'IANA (Internet Assigned Numbers Authority) et les RIR (Regional Internet Registries).<br/></p>
