# 1983 - Adoption du protocole TCP/IP

<div class="imported-content">

<div class="card">





<div><a class="relative-url" href="https://commons.wikimedia.org/wiki/File:TCPIP_couche_ISO_modele_OSI.png" rel="noopener" target="_blank"><p><img class="imported-image ptSynthesisImg" data-original-url="media%2FpearlContent%2F551773976" id="l_2360809" loading="lazy" pearltrees="true" src="/Manuel-SNT/assets/imported/internet-reperes-1983-adoption-du-protocole-tcp-ip-html/image-01.jpg" width="495"/></p></a></div><div id="note-text"><p><br/></p>
<p>Le protocole TCP (Transmission Control Protocol) / IP (Internet Protocol) est adopté comme standard pour les communications sur ARPANET. Il fournit une méthode de transmission de données fiable et établit la base de l'architecture du réseau Internet.</p>
