# 1983 - Adoption du protocole TCP/IP

<div class="imported-content">

<p><br/></p>
<p>Le protocole TCP (Transmission Control Protocol) / IP (Internet Protocol) est adopté comme standard pour les communications sur ARPANET. Il fournit une méthode de transmission de données fiable et établit la base de l'architecture du réseau Internet.</p>
