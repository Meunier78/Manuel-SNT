<p class="scrap-without-first-image"></p><div id="note-text"><p>Lorsque l'internaute entre dans l'URL une adresse symbolique de type www<a href="http://www.monsite.fr" rel="noopener" target="_blank">.</a>pearltrees.com, sa machine se connecte d'abord à un serveur de nom de domaine pour obtenir l'adresse IP correspondante. Ensuite, les routeurs permettent d'acheminer la requête, de réseaux en réseaux vers le serveur hébergeant le site recherché.</p>
<p>Nous allons voir comment un programme peut retrouver une adresse IP à partir d’une adresse symbolique, mais aussi l'identifiant réseau et le numéro d’hôte à partir de l'adresse IP.</p>
