<p class="scrap-without-first-image"></p><div id="note-text"><p>On souhaite afficher un nombre en format décimal, en format binaire et en format hexadécimal. Vous devez implémenter les trois fonctions suivantes :<br/></p>
<ul><li>dec_vers_binhex, qui prend un nombre décimal en paramètre et qui renvoie le code binaire et hexadécimal.<br/></li><li>bin_vers_dechex, qui prend un nombre binaire en paramètre et qui renvoie le code décimal et hexadécimal.<br/></li><li>hex_vers_decbin, qui prend un nombre hexadécimal en paramètre et qui renvoie le code décimal et binaire.<br/></li></ul><p>Pour convertir un nombre entier décimal  en une chaine de caractère binaire et hexadécimal, on utilise les méthodes <b><i>bin(nb_dec)</i></b> et <b><i>hex(nb_dec)</i></b>.</p>
<p>Pour convertir une chaine de caractères codée dans une base quelconque en entier décimal, on utilise la méthode <b><i>int(chaine, base)</i></b> avec le paramètre base valant 2, 8, 10 ou 16. </p>
