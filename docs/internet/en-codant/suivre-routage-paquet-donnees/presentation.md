<p class="scrap-without-first-image"></p><div id="note-text"><p>La commande ping suivie d'une adresse IP ou d'un nom de domaine permet de savoir s'il est possible d'établir une connexion entre deux machines.<br/></p>
<p>Nous partons du réseau d'établissement partiel que nous avons modélisé afin de simuler son fonctionnement. L'utilisateur donne l'adresse IP ou le nom de domaine après la commande ping et le programme renvoie le chemin emprunté du poste source vers sa destination.</p>
