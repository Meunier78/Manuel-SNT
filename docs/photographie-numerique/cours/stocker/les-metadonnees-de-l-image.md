# Les métadonnées de l'image

<div class="imported-content">

<p>Les fichiers des photographies numériques contiennent aussi des métadonnées au format EXIF (Exchangeable Image File Format). <br/></p>
<p>Ces données peuvent préciser, par exemple :<br/></p>
<ul><li>le modèle de l’appareil ; <br/></li><li>la localisation GPS de la prise de vue ;  <br/></li><li>le temps de pose (ou d'exposition) de la photo ;<br/></li><li>le type de l'objectif et son ouverture ;<br/></li><li>la distance de mise au point ;<br/></li><li>l'auteur, le copyright, etc. </li></ul>
