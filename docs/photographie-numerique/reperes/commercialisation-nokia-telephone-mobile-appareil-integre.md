# 2002 - Commercialisation du Nokia 7650, téléphone mobile avec appareil photo intégré

<div class="imported-content">

<p>En 2002 est commercialisé le Nokia 7650, l'un des premiers téléphones mobiles avec appareil photo intégré. Sa définition était de 640×480 pixels.<br/></p>
