# Le format EXIF

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Les fichiers stockant des photographies numériques contiennent des métadonnées au format EXIF (Exchangeable Image File Format). Ces données peuvent indiquer, par exemple :</p>
<ul><li>le modèle de l’appareil ;</li><li>la localisation GPS de la prise de vue ;  </li><li>le temps de pose de la photo ;</li><li>le type d'objectif et son ouverture ;</li><li>la distance de mise au point ;</li><li>l'auteur, le copyright, etc.<br/><br/></li></ul></div>
