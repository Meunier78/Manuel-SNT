# Des informations importantes cachées dans les fichiers image

<div class="imported-content">

<p>À l'époque de la photographie argentique, les photographes notaient parfois dans un carnet :</p>
<ul><li>l'endroit où ils avaient pris chaque photographie ;</li><li>l'appareil photo utilisé (surtout s'ils en possédaient plusieurs) ;</li><li>l'objectif utilisé (les objectifs étaient souvent interchangeables) ;</li><li>le temps de pose de la prise de vue, c'est-à-dire le temps pendant lequel l'appareil photo capturait la lumière (quelques millièmes de seconde, quelques centièmes de seconde, une fraction de seconde).<br/></li></ul><p>Aujourd'hui, ces informations et bien d'autres sont automatiquement incluses dans le fichier de l'image numérique.</p>
