# Les métadonnées GPS

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Les données EXIF indiquent souvent la localisation "GPS" de la prise de vue. </p>
<p>Avec elles, on peut récupérer les valeurs de longitude et de latitude du lieu. </p>
<p>La latitude et la longitude, leur affichage et leur conversion, sont traitées dans un autre chapitre. Mais nous vous proposons quelques éléments permettant de vous approprier ces notions ou vous les remémorer.<br/></p>
<p>Vous pourrez ensuite identifier les endroits où les photos de cette activité ont été réalisées ! </p>
