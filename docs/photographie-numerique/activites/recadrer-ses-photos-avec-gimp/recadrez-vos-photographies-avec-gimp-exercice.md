# Recadrez vos photographies avec Gimp - Exercice

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>À vous de choisir deux photographies que vous avez vous-mêmes réalisées ou que vous avez trouvées sur internet. </p>
<p>Pour chacune, appliquez un recadrage esthétique, avec redressement de l'horizon si nécessaire. </p>
<p>Changez les proportions de ces deux photographies. Transformez la première au format 4:3, la seconde au format 3:2. </p>
