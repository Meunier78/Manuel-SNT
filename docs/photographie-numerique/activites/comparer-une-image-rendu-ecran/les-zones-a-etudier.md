# Les zones à étudier

<div class="imported-content">

<p class="scrap-without-first-image"></p><p>Avec le logiciel sélectionné, observez chacune des zones suivantes : </p>
<p>1. le centre de la fleur ;<br/></p>
<p>2. le centre d'un pétale ; </p>
<p>3. l'arrière plan de la fleur ; </p>
<p>4. le contenu d'une lettre du mot "Galerie".</p>
