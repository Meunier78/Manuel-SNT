# Zoomer avec la souris

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Nous disposons d'une capture d'écran du téléphone portable, sous la forme d'un fichier. Cette image numérique de l'écran est stockée sous la forme de valeurs numériques. Il est possible d'observer ce fichier avec un logiciel éditeur d'image. <br/></p>
<p>L'image est constituée de 720 pixels de large sur 1560 pixels de haut. <br/></p>
<p>Chacun de ces pixels possède une couleur, composée de trois valeurs : rouge, vert bleu. </p>
<p>1. Téléchargez le fichier intitulé "Capture d'écran à observer de près". <br/></p>
<p>2. Ouvrez le fichier avec un logiciel ou une application qui permet d'analyser la composition des pixels :<br/></p>
<ul><li>Sur un ordinateur PC, utilisez les logiciels intégrés Paint ou Paint 3D qui possèdent chacun un sélecteur de couleur. Il s'agit de l'icône "compte-goutte" ou "pipette". <br/></li><li>Sur un téléphone portable ou sur une tablette, c'est également possible en installant une application du type "color picker". Effectuez une recherche dans le magasin d'applications avec ces mots clés. </li></ul><p>3. Une fois l'image ouverte dans le logiciel ou l'application, relevez les valeurs "rouge", "vert" et "bleu" de pixels situés dans les zones à étudier (voir image).</p>
