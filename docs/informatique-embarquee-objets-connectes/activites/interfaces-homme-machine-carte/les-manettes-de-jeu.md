# Les manettes de jeu

<p>Les manettes de jeu ont évolué depuis les simples boutons et joysticks qui constituaient les premières générations. Les croix directionnelles et des boutons supplémentaires ont été introduits dans les années 1980, suivis des gâchettes analogiques, de vibrations et de capteurs de mouvement. Les manettes modernes sont très avancées, avec des fonctionnalités telles que des écrans tactiles et des détecteurs de mouvement sophistiqués. </p>
<p>Une manette de jeu est une interface homme-machine (IHM) car elle permet à un utilisateur de communiquer avec une machine ou avec un système informatique.</p>
