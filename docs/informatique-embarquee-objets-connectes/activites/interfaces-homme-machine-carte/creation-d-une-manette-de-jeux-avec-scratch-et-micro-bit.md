# Création d'une manette de jeux avec Scratch et micro:bit

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Nous allons utiliser une micro:bit, pour contrôler un jeu de raquettes simplifié. La carte micro:bit sera utilisée comme une manette de jeux à deux boutons.  </p>
<p>Cette activité nécessite : <br/></p>
<ul><li>le logiciel Scratch installé sur un ordinateur (<a href="https://scratch.mit.edu/download" rel="noopener" target="_blank">https://scratch.mit.edu/download</a> pour l'installer) ; </li><li>une carte micro:bit reliée à Scratch (suivre les instructions au <a href="https://scratch.mit.edu/microbit)" rel="noopener" target="_blank">https://scratch.mit.edu/microbit)</a>. </li></ul><p>Nous allons nous inspirer du <a href="https://fr.wikipedia.org/wiki/Pong" rel="noopener" target="_blank">jeu Pong</a>, créé dans les années 1970.</p>
