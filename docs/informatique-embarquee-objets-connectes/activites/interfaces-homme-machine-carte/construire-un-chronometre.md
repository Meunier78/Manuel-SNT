# Construire un chronomètre

<div class="imported-content">

<p>L'objectif est ici de construire un chronomètre simple : le bouton de gauche (A) de la micro:bit déclenche le chronomètre en affichant un "zéro" sur la matrice 5x5 (ou tout autre signe, une croix, un "+", etc.) comme signal de déclenchement. Ensuite, le bouton de droite (B) provoque l'affichage du nombre de seconde écoulées depuis le déclenchement du chronomètre.<br/></p>
