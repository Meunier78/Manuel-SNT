# Étapes pour construire un chronomètre - Activité

<div class="imported-content">

<p>Pour construire le chronomètre :<br/></p>
<ul><li>Rendez-vous sur le site <a href="https://fr.vittascience.com/microbit" rel="noopener" target="_blank">https://fr.vittascience.com/microbit</a>/ afin d'accéder à l'interface de programmation.<br/></li><li>Dans le bloc "répéter indéfiniment", ajoutez le bloc "si le bouton A est appuyé, alors" (de la rubrique "entrées / sorties"). </li><li>Imbriquez les blocs "afficher 0" (dans la rubrique "affichage") et "démarrer le chronomètre" (de la rubrique "entrées / sorties").<br/></li><li>Au même niveau que le bloc "si le bouton A est appuyé, alors" , placez un nouveau bloc "si le bouton B est appuyé, alors" (en changeant "A" en "B"). <br/></li><li>Glissez le bloc "valeur du chronomètre en ms" (de la rubrique "entrées / sorties").</li></ul><p>On peut aussi s'assurer du lancement du chronomètre au démarrage du programme : <br/></p>
<ul><li> Insérez donc le bloc "démarrer le chronomètre" (de la rubrique "entrées / sorties") dans le bloc "Au démarrage". </li></ul><p>C'est le moment de tester le programme dans l'interface de programmation. Cliquez sur la flèche "démarrer le simulateur" en haut à droite de l'écran.  </p>
<p>Reste à téléverser le programme sur une vraie micro:bit, si vous en avez une à disposition. Pour cela, branchez la micro:bit sur le port USB de l'ordinateur, puis cliquez sur "téléverser". Le chronomètre devrait maintenant fonctionner sur la carte micro:bit. <br/></p>
