# BBC micro:bit

<div class="imported-content">

<p>Le BBC micro:bit est un ordinateur de poche qui vous initie à la façon dont le logiciel et le matériel fonctionnent ensemble. Il dispose d'un afficheur LED, de boutons, de capteurs et de nombreuses fonctionnalités d'entrée/sortie qui, une fois programmées, permettent d'interagir avec vous et votre univers.</p>
