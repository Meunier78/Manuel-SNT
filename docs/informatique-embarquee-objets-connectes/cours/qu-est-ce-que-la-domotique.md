# Qu'est-ce que la domotique ?

<div class="imported-content">

<p>La domotique est un domaine où les objets connectés sont fréquents. La domotique est un ensemble de technologies et de systèmes visant à automatiser et à contrôler les équipements d'un bâtiment, tels l'éclairage, le chauffage, la climatisation, les volets roulants, la sécurité, la communication, etc. <br/></p>
