# Soyons vigilants !

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>Un accès non maîtrisé à vos objets connectés peut représenter un risque de sécurité ou d'atteinte à votre vie privée. </p>
<p>Comme le conseille le site <a href="https://www.economie.gouv.fr/dgccrf/Publications/Vie-pratique/Fiches-pratiques/objets-connectes" rel="noopener" target="_blank">https://www.economie.gouv.fr/dgccrf/Publications/Vie-pratique/Fiches-pratiques/objets-connectes</a>, il est prudent de : </p>
<ul><li>s'informer sur les caractéristiques de l'objet, son fonctionnement, ses interactions avec les autres appareils électroniques et, le cas échéant, sur les précautions à prendre ; </li><li>procéder régulièrement aux mises à jour logicielles ; </li><li>changer le nom et/ou le mot de passe par défaut de votre objet connecté ; </li><li>limiter l'accès de votre objet connecté par d'autres appareils ou d'autres utilisateurs.</li></ul><p><img ="xl"="" class="imported-image" data-height="1000" data-original-url="logo%2Fhash%2F551776714%3Fh%3D11a4ca1552f5a17f81124f71570a8889" data-width="1000" id="l_2326178" loading="lazy" src="/Manuel-SNT/assets/imported/informatique-embarquee-cours-soyons-vigilants-html/image-01.jpg"/></p>
