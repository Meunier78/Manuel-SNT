# Pour le meilleur

<div class="imported-content">

<div class="card">




<p class="scrap-without-first-image"></p><div id="note-text"><p>Individuellement ou en groupe, recherchez des applications récentes ou futures de systèmes informatiques embarqués qui amènent, selon vous, un réel progrès.<br/></p>
<p>Ces applications peuvent se situer dans différents domaines : environnemental, médical, culturel, etc.<br/></p>
<p>Pour chacune de ces applications, présentez le progrès qui vous semble intéressant, en justifiant votre avis.<br/></p>
<p>Nous vous proposons quelques exemples susceptibles de vous inspirer. </p>
