# Fabriquer un thermomètre avec une carte micro:bit - Exercice

<div class="imported-content">

<p>L'objectif de cet exercice est de fabriquer un thermomètre avec une carte micro:bit.<br/></p>
<p>Celle-ci contient un capteur de température qui mesure la température en degrés celsius, en degrés fahrenheit ou en kelvin.<br/></p>
<p>La carte micro:bit doit afficher la température ambiante toutes les trois secondes. La valeur affichée défilera à l'écran.  </p>
<p>Utilisez le site </p>
<p><a href="https://fr.vittascience.com/microbit/?mode=blocks&amp;console=bottom&amp;toolbox=vittascience" rel="noopener" target="_blank">https://fr.vittascience.com/microbit/?mode=blocks&amp;console=bottom&amp;toolbox=vittascience</a> pour programmer l'affichage de la température qui défilera sur l'écran de la carte micro:bit. </p>
<p>L'affichage doit se répéter indéfiniment, toutes les trois secondes. </p>
