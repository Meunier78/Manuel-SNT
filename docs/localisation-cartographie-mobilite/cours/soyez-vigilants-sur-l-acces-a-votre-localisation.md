# Soyez vigilants sur l'accès à votre localisation

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>Des applications et des programmes peuvent accéder à votre localisation sans que vous le souhaitiez. Aujourd’hui, les paramètres d’accès à cette localisation peuvent être réglés, mais il convient d’être vigilant. <br/></p>
<p>La capture d'écran d'un téléphone, ci-jointe, présente quelques applications pouvant accéder à sa localisation GPS. Pour certaines applications, la localisation semble logique. Pour d'autres, elle peut interroger. </p>
<p>Aussi, comment ces applications utilisent-elles ces données GPS ? S'appuient-elles sur cette information pour prioriser des recommandations ? Partagent-elles ces données avec des tiers ?<br/></p>
<p>Les conditions générales d'utilisation (CGU) des applications devraient pouvoir vous renseigner. Encore faut-il prendre le temps de les lire avec attention et de mesurer l'intérêt de les laisser accéder à la localisation de l'utilisateur. </p>
