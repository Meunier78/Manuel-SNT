# Les données de géolocalisation dans les images (EXIF)

<div class="imported-content">

<p><img ="xl"="" class="imported-image ptSynthesisImg" data-height="1333" data-original-url="media%2FpearlContent%2F551775564" data-width="1000" id="l_2436681" loading="lazy" src="/Manuel-SNT/assets/imported/localisation-cartographie-cours-les-donnees-de-geolocalisation-dans-les-images-exif-html/image-01.jpg"/></p>
<p>Surtout lorsqu'elles sont réalisées par un téléphone portable, la plupart des photographies contiennent des métadonnées EXIF. Dans celles-ci, on peut récupérer le lieu et l'heure du cliché.</p>
<p>En téléchargeant la photo ci-jointe, (cliquez dessus, pour l'ouvrir dans un onglet, puis clic droit / enregistrer), on peut identifier la localisation de la prise de vue. <br/></p>
<p>On s'aperçoit que la photographie a été prise à cet endroit : <a href="https://www.openstreetmap.org/search?query=52%C2%B031%2720.5%22N%2013%C2%B024%2730.7%22E#map=13/52.5224/13.4084" rel="noopener" target="_blank">https://www.openstreetmap.org/search?query=52%C2%B031%2720.5%22N%2013%C2%B024%2730.7%22E#map=13/52.5224/13.4084</a> en mai 2022. </p>
