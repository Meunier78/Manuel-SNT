# 2008 - L'iPhone connait et affiche sa position avec l'ajout d'un capteur GPS

<p>En introduisant un capteur GPS dans ses iPhones, Apple permet de développer de nouveaux usages en mobilité. Par exemple, le téléphone ajoute la localisation aux images prises par la caméra. Les recherches sur internet peuvent prendre en compte la position de l'utilisateur. Cela rend également possible le partage de position par les applications de réseaux sociaux. Surtout, le téléphone devient un moyen de s'orienter dans l'espace, à pied, en voiture, en vélo. <br/></p>
