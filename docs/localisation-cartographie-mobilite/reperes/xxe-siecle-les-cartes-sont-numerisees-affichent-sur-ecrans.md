# XXe siècle - Les cartes sont numérisées et s'affichent sur les écrans

<div class="imported-content">

<div class="card">





<div><a class="relative-url" href="https://oldpcgaming.net/microsoft-encarta-99-review/" rel="noopener" target="_blank"><div id="note-text"><p><img ="xl"="" class="imported-image ptSynthesisImg" data-height="537" data-original-url="logo%2Fhash%2F551775481%3Fh%3D6b740e7b44ab06eaca84ad856ad467de" data-width="750" id="l_2316064" loading="lazy" src="/Manuel-SNT/assets/imported/localisation-cartographie-reperes-xxe-siecle-les-cartes-sont-numerisees-affichent-sur-ecrans-html/image-01.jpg"/></p></div></a></div>
<p>Les cartes sont peu à peu numérisées, et à la fin du XX<sup>e</sup> siècle, elles font leur apparition sur les écrans des ordinateurs. Des versions grand public sont commercialisées sur différents supports (CD, DVD). Microsoft a diffusé un atlas mondial intitulé "Atlas Mondial Encarta" qui a popularisé les versions numériques des cartes du globe. <br/></p>
