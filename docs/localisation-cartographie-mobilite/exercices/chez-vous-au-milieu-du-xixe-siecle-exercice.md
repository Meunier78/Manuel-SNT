# Chez vous, au milieu du XIXe siècle - Exercice

<div class="imported-content">

<div class="card">




<div id="note-text"><p class="scrap-without-first-image"></p><p>Il s'agit d'utiliser le site <a href="http://www.geoportail.gouv.fr" rel="noopener" target="_blank">www.geoportail.gouv.fr</a> pour afficher les cartes d'État-major <br/>correspondant aux environs de votre domicile. </p>
<p>Pour cela, une fois la carte bien centrée sur l'endroit où vous habitez (ou sur un autre endroit de votre choix) : <br/></p>
<ul><li>cliquez sur le bouton "sélection de couches" ;<br/></li><li>cliquez sur sur le bouton "+ de données" ;<br/></li><li>activez "voir tous les fonds de cartes" ;<br/></li><li>sélectionnez "carte de l'État-major (1820-1866)". <br/></li></ul><p>Parvenez-vous à identifier des noms de lieux connus ? Repérez-vous des différences notables avec la géographie actuelle ?  </p>
