<p>Si vous avez oublié votre mot de passe, cliquez sur « Vous avez déjà un compte ? » dans le coin supérieur droit, puis sur « Vous avez oublié votre mot de passe ? ». Entrez l'adresse électronique qui est connectée à votre compte, et nous vous enverrons un e-mail avec un lien pour réinitialiser votre mot de passe ! N'oubliez pas de vérifier votre boîte spam, il pourrait s'y retrouver.</p>
<p>Si vous ne recevez pas d'e-mail, vous avez peut-être activé la connexion via Google/Facebook. Si tel est le cas, envoyez un e-mail à <a href="mailto:info@geoguessr.com">info@geoguessr.com</a> et nous vous aiderons pour la suite. Si vous avez oublié avec quelle adresse électronique vous vous êtes inscrit, pas de soucis ! Contactez-nous à <a href="mailto:info@geoguessr.com">info@geoguessr.com</a> et nous vous aiderons !</p>
