# Un exemple de voyage présenté avec Umap

<div class="imported-content">

<p>Voici l'exemple d'un trajet en train au départ de Paris. </p>
<p>Les voyageurs traverseront l'Allemagne puis le Danemark, pour revenir en France <i>via </i>l'Allemagne. <br/></p>
<p>La carte est accessible ici : <a href="http://umap.openstreetmap.fr/fr/map/vacances-en-train-france-allemagne-et-danemark_895977" rel="noopener" target="_blank">http://umap.openstreetmap.fr/fr/map/vacances-en-train-france-allemagne-et-danemark_895977</a><br/></p>
<p>Elle a été crée sur Umap et peut être partagée avec d'autres utilisateurs. </p>
