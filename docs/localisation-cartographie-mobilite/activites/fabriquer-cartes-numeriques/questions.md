# Questions

<p>1. Comparez les photographies aériennes du lycée international de Palaiseau extraites de trois sites différents en mars 2023 : Google Maps, Geoportail, Bing Maps. Qu'observe-t-on ? <br/></p>
<p>2. Sur les mêmes sites, affichez les photographies aériennes de votre domicile ou d'un lieu que vous connaissez bien. Comparez les clichés, proposez des hypothèses sur les dates auxquelles les différentes photographies ont été réalisées. Peut-on identifier la période de l'année (saison) correspondante ?<br/></p>
