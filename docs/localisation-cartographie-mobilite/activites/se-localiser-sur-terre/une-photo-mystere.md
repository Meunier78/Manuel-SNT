# Une photo mystère

<div class="imported-content">

<p>Sur son ordinateur, Isabelle retrouve une vieille photographie d'une tortue de mer datant d'avril 2014. Elle ne parvient pas à se rappeler où elle l'a prise. <br/></p>
<p>En affichant ses propriétés dans l'explorateur de fichiers, elle obtient les informations suivantes : </p>
<ul><li>latitude : 46° 9' 11.7235'' N ;</li><li>longitude : 1° 9' 2.5340'' W ;</li><li>altitude : 4,942 m.</li></ul>
