# Présentation

<div class="imported-content">

<p>Pour surveiller l'état des cultures, les agriculteurs recourent parfois à l'utilisation d'un drone. À partir des données enregistrées durant le vol, nous allons reconstituer le parcours du drone et l'afficher sur une carte de type "open street map". Nous en déduirons également la vitesse maximale atteinte par le drone lors de ce vol. <br/></p>
<p>La vidéo, prise par le drone, contient un fichier de sous-titres avec diverses informations actualisées pendant le vol : xxx, xxx. Nous avons converti ce fichier au format csv pour faciliter son exploitation. </p>
